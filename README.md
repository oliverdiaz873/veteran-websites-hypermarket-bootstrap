# Veteran Websites – Hypermarket (Bootstrap Edition)

A veteran-level hypermarket website built with **HTML**, **Bootstrap 5**, **CSS**, and **JavaScript**.

---

## Project Overview

This project represents a **veteran-level front-end website**, marking the progression from
**Vanilla HTML/CSS/JavaScript** to **framework-based development** using Bootstrap.

Instead of focusing on building everything from scratch, this version prioritizes
**efficiency, scalability, and standardized UI patterns**, following professional front-end workflows.

It belongs to the `veteran-websites` category and demonstrates the ability to work confidently
with established UI frameworks.

---

## Features

- Fully responsive layout using **Bootstrap’s 12-column grid system**.
- Component-driven interface built with:
  - Navbars
  - Cards
  - Modals
  - Carousels
  - Dropdowns
- Mobile-first design using Bootstrap breakpoints.
- Reduced custom CSS through extensive use of **Bootstrap utility classes**.
- JavaScript logic integrated with **Bootstrap’s JavaScript API**.
- Accessible components with built-in **ARIA roles and keyboard navigation**.
- SEO-friendly meta tags and semantic markup.
- Clean, scalable, and maintainable project structure.

---

## Technologies Used

- **HTML5** – Semantic and accessible markup.
- **Bootstrap 5** – UI components, grid system, and utilities.
- **CSS3 (Custom)** – Minor overrides and brand customization.
- **JavaScript (ES6+)** – Interactivity and framework integration.

---

## Learning Goals

This project helps practice and reinforce:

- Working with UI frameworks in real-world scenarios.
- Migrating from custom layouts to framework-based systems.
- Component-based front-end development.
- Utility-first styling approaches.
- Building scalable, maintainable front-end projects.
- Preparing for modern JavaScript frameworks.
